## Objective
Build a machine learning model that classifies the sentiment of IMDB movie reviews as positive or negative, providing insights into public opinion.

## Files
| File | Description |
|------|-------------|
| `ParamKoli_L1_Task4.ipynb` | Main Jupyter Notebook with full analysis |
| `IMDB Dataset.csv` | Dataset used for analysis |

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, NLTK, WordCloud, Jupyter Notebook

## Analysis Performed
- Dataset loading and inspection
- Sentiment distribution visualisation
- Text preprocessing pipeline
- TF-IDF feature extraction
- Naive Bayes classifier
- Logistic Regression classifier
- Confusion matrices for both models
- WordCloud visualisation for positive and negative reviews
- Error analysis on misclassified examples
- Model comparison table

## Best Model
Logistic Regression — outperformed Naive Bayes across all metrics
