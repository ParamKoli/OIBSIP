## 🚀 How to Run Any Project

1. **Clone this repository:**
   ```bash
   git clone https://github.com/ParamKoli/OIBSIP 
   ```

2. **Install dependencies:**
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn nltk textblob jupyter
   ```

3. **Navigate to any task folder:**
   ```bash
   cd DataAnalytics-L1-EDARetailSales
   ```

4. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

5. Open the `.ipynb` file and run all cells.
