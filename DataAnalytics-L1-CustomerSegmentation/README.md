**Name:** Param Parag Koli
**Track:** Data Analytics
**Level:** 1 | **Task:** 2
**Batch:** July 2026

## Objective
Apply clustering algorithms to segment an e-commerce company's customer base into distinct groups based on purchasing behaviour, enabling targeted marketing strategies.

## Files
| File | Description |
|------|-------------|
| `ParamKoli_L1_Task2.ipynb` | Main Jupyter Notebook with full analysis |
| `data.csv` | Dataset used for analysis |

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Descriptive statistics
- RFM Feature Engineering
- Data normalisation using StandardScaler
- Elbow Method to find optimal K
- K-Means Clustering
- Cluster visualisation
- Cluster profiling and customer segmentation

## Business Recommendations
1. Focus retention efforts on Loyal customers
2. Re-engage at-risk customers with win-back campaigns
3. Nurture potential loyalists into high-value customers
