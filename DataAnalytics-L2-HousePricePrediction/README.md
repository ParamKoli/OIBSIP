## Objective
Build and evaluate a linear regression model that predicts house prices based on features such as area, location, number of rooms, and age.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## Analysis Performed
- Dataset loading and inspection
- Price distribution visualisation
- Categorical feature encoding
- Correlation heatmap
- Train/test split
- Linear Regression model
- Ridge and Lasso regularised models
- Actual vs Predicted scatter plots
- Residual plot
- Coefficient analysis
- Model comparison table

## Best Model
Linear Regression — simple, interpretable, and performed comparably to regularised models
