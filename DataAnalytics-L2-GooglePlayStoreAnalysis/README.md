## Objective
Perform a comprehensive data analysis of the Google Play Store ecosystem — cleaning messy real-world data, exploring app categories, analysing ratings and pricing trends, and conducting sentiment analysis on user reviews.

## Files
| File | Description |
|------|-------------|
| `ParamKoli_L2_Task4.ipynb` | Main Jupyter Notebook with full analysis |
| `googleplaystore.csv` | Apps dataset |
| `googleplaystore_user_reviews.csv` | User reviews dataset |

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, TextBlob, Jupyter Notebook

## Analysis Performed
- Dataset loading and inspection
- Data cleaning and type correction
- Category analysis — most popular and most installed categories
- Ratings distribution and average rating by category
- App size vs installs correlation
- Pricing analysis — free vs paid apps
- Sentiment analysis on user reviews using TextBlob
- Sentiment distribution by app category

## Business Recommendations
1. Launch in underserved but high-install categories
2. Keep apps lightweight to improve download rates
3. Monitor and improve sentiment in Game and Family categories
